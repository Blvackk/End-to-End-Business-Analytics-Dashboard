# 🍔 Food Delivery Business Analytics Dashboard | Power BI

An interactive Business Intelligence dashboard developed using **Power BI** to analyze food delivery business performance. The dashboard provides actionable insights into revenue, customer behavior, restaurant performance, and delivery operations using interactive visualizations and DAX measures.

---

## 📌 Project Overview

This project transforms raw food delivery data into an interactive dashboard that enables stakeholders to monitor business performance and make data-driven decisions.

The dashboard focuses on:

- Revenue Analysis
- Customer Analytics
- Restaurant Performance
- Delivery Operations
- Dynamic KPI Analysis

---

## 🚀 Features

- Interactive and user-friendly dashboard
- Dynamic KPI selection using DAX
- Drill-down and drill-up functionality
- Cross-filtering between visuals
- Date-based trend analysis
- Business KPI monitoring
- Responsive visualizations
